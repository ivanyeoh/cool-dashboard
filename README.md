## Getting Started

```bash
cp .env.sample .env
npm i
npm run mockapi
npm run dev
```

### User Readonly page

Open [http://localhost:3000](http://localhost:3000)

### Admin Configuration page

Open [http://localhost:3000/admin](http://localhost:3000/admin)
